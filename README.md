Medieval Mayhem is a 2D two player fighting game. 

Objective: Kill Opponent

Controls:
  
  Player 1:
    Move LEFT: A
    Move RIGHT: D
    JUMP: W
    ATTACK 1: R
    ATTACK 2: T
  
  Player 1:
    Move LEFT: LEFT ARROW KEY
    Move RIGHT: RIGHT ARROW KEY
    JUMP: UP ARROW KEY
    ATTACK 1: N
    ATTACK 2: M
